# Sales_trend_analysis
Project Overview
This project analyzes sales data to understand monthly trends, product performance, and regional sales distribution using Python.

Objective
The objective of this project is to analyze sales patterns over time and identify top-performing products and regions.

Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook

Dataset
The dataset contains sales records with the following fields:
Date
Product
Region
Sales
Quantity

Analysis Performed
Imported and cleaned the sales dataset
Converted date column for time-based analysis
Performed monthly sales aggregation
Analyzed product-wise and region-wise performance
Visualized sales trends using graphs

 Results
Sales show an increasing trend over months
Certain products generate higher revenue compared to others
Some regions perform better in terms of total sales
Sales vary depending on time period and demand

 Visualization
A line chart was used to represent monthly sales trends, and bar charts were used for product-wise and region-wise comparisons.

 Conclusion
This analysis provides insights into sales performance across time, products, and regions. It helps in understanding business trends and supports better decision-making for improving sales strategy.
